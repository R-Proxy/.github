# 🔄 R-Proxy

Simplified Reverse Proxy Management.

---

## ✨ Features

- Easy install with Docker 🐳
- Effortless Dashboard Management 🖥️
- Supports HTTP, HTTPS, TCP & UDP 🌐
- Fully Opensource 🛠️
